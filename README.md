# 🌦️ WeatherX Premium

A modern, premium weather web application inspired by Apple Weather and Nothing OS. WeatherX Premium provides real-time weather updates, live location support, interactive maps, air quality monitoring, and a beautiful responsive interface.

---

## ✨ Features

- 📍 Live GPS Location Detection
- 🔎 Smart City Search with Suggestions
- 🇮🇳 India-focused City Search
- 🌡️ Real-Time Weather Information
- 🌤️ Animated Weather Icons
- 🌬️ Wind Speed & Direction
- 💧 Humidity
- 👀 Visibility
- 🌅 Sunrise & Sunset
- 🌙 Day/Night Detection
- ☀️ UV Index
- 🌫️ Air Quality Index (AQI)
- 📅 5-Day / 7-Day Forecast
- 🕒 Hourly Weather Forecast
- 🗺️ Interactive Leaflet Map
- ⭐ Favorite Cities
- 🕘 Recent Searches
- 📱 Fully Responsive Design
- 🌧️ Dynamic Weather Effects
- 🌙 Dark Theme
- ⚡ Smooth Animations
- 💎 Glassmorphism UI
